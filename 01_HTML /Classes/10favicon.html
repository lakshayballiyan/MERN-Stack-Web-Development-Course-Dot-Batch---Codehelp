WHAT IS FAVICON AND HOW TO ADD THE FAVICON ?



A favicon is a small image displayed next to the page title in the browser tab.A favicon image is displayed to the left of the page title in the browser tab



HOW TO ADD ?



To add a favicon to your website, either save your favicon image to the root directory of your webserver, or create a folder in the root directory called images, and save your favicon image in this folder.
