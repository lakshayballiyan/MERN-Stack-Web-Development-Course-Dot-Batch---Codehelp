TYPES OF LIST :

LIST ARE OF THREE TYPES : ORDERED LIST , UNORDERED LIST AND DESCRIPTION LIST..



ORDERED LIST = items in a ordered way


  1. MILK
  2. PANEER
  3. PEANOUT BUTTER
  4. COLD DRINK
  5. BISCUIT

UNORDERED LIST = items in a unordered way


  • COFFEE
  • TEA
  • FRUITS
  • JUICES
  • VEGETABLES

DESCRIPTION LIST = a description list is a list of terms , with a description of each terms

dl = description list , dt = defines the terms , dd = describe each term
